1. Download ffmpeg from their main site.
```
https://ffmpeg.org/
```

2. Select the platform/OS you use.


3. For Windows, add the bin to the environmental paths.

4. Convert your video using the following command in the terminal.
```bash
ffmpeg -i <video_name.ext> <outputname.gif>
```


5. Can modify the output even more by looking up the documentations.
